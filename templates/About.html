Bad Credit Score Doesn't Make a Bad Borrower

Goal

The web application allows a lender to do a comprehensive analysis of a loan application. The app doesn't classify whether a borrower is safe or not. It is intended to present a more detailed analysis of a loan applicant without looking at the credit score.

Data and Model

Loan assessment is done based on an underlying machine learning model. Actual model linked in the github page uses a random forest model to predict the probability to default and create visualizations. Due to memory limitation, current app uses a simple logistic regression model to predict the score.

The data that is trained using the data provided by Lending club. The dataset has about 150 attributes for each loan. The trained model uses data from 2007-2017.

How it works?

Go to home page and fill out the information on the form. Most of the fields can be found in application form. There are few fields which are properties of a loan that is provided to the customer (such as interest rate, grade and subgrade). On hitting a submit button, a new window will appear with a score for probability to default. There are other three tabs, all of which presents a visualization that will give more insights regarding the customer.
